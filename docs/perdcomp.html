Antes de continuar
Esta ferramenta realiza análises automatizadas com base nas informações extraídas dos arquivos enviados, com objetivo de apoiar processos de revisão, compliance e identificação de inconsistências tributárias.

Os resultados apresentados possuem caráter auxiliar e não substituem validações técnicas, revisões manuais ou pareceres especializados.

Recomenda-se que todas as informações e conclusões sejam avaliadas pela equipe fiscal, tributária ou jurídica responsável antes de qualquer tomada de decisão ou transmissão perante os órgãos competentes.

Os arquivos enviados são processados localmente e não são armazenados em servidores externos.

Análise de PER/DCOMP

Execute o script no eCAC para baixar os PDFs, depois faça o upload e analise.

⚡ Passo 1 — Baixar os PDFs do eCAC
1 No eCAC, acesse: Restituição e Compensação → PERDCOMP Web → Visualizar Documentos → Documentos Entregues e filtre o período desejado
2 Pressione F12 para abrir o DevTools → clique na aba Console
3 No dropdown do Console (mostra top), selecione o contexto www3.cav.receita.fazenda.gov.br
4 Digite allow pasting e pressione Enter para liberar o console
5 Clique em Copiar script abaixo, cole no console e pressione Enter — o script percorre todas as páginas e baixa cada PDF automaticamente
📁 Passo 2 — Fazer upload dos PDFs baixados
Selecione todos os PDFs que foram baixados pelo script (ou manualmente no eCAC).
🔍 Passo 3 — Analisar
Extrai os dados de todos os PDFs e executa a análise de compliance automaticamente.
