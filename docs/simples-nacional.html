Fonte dos dados — Validação recomendada
Os dados são obtidos via API pública da Receita Federal (BrasilAPI / ReceitaWS) e refletem a última atualização disponível nos servidores públicos. Para empresas com múltiplos ciclos de entrada e saída do Simples, o histórico anual é calculado a partir das datas registradas na base mais recente — recomenda-se confirmação direta no Portal do Simples Nacional (www8.receita.fazenda.gov.br) para casos críticos.
